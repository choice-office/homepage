## 2. 행정사 사무소용 재구성

법무법인 → **행정사 사무소**로 핵심 차이: 소송/변호가 아니라 **관공서 행정 절차 대행**이 중심.
업무분야를 행정사 실제 업무로 치환하고, "성공사례"는 "처리사례", 신뢰지표(경력/처리건수)를 강조.

### 2-1. 재구성 IA / 네비게이션

| 대메뉴 | 하위 메뉴 | 비고 |
|---|---|---|
| **사무소 소개** | 인사말 · 사무소 강점 · 오시는 길 | lawus "로어스 소개" 대응 |
| **구성원** | 행정사 소개 | 대표/소속 행정사 |
| **업무분야** | 출입국·체류/비자 · 인허가·신고 · 행정심판·이의신청 · 토지·건축 · 법인설립·등록 · 손실보상·민원 (확장 가능) | lawus 업무분야 대응 |
| **처리사례** | 사례 목록 + 상세 | lawus "성공사례" |
| **상담후기** | 의뢰인 후기 | (선택) |
| **온라인 상담** | 문의 폼 + 카카오톡 + 전화 | lawus "온라인 상담" |

### 2-2. 행정사 업무분야 (카드 6종, 확장 가능)

1. **출입국·체류 / 비자** — 체류자격 변경·연장, 비자, 귀화·영주, 초청
2. **각종 인·허가 / 신고** — 영업·건축·식품 인허가, 등록·신고 대행
3. **행정심판 / 이의신청** — 처분 취소·이의신청, 심판청구서 작성·대리
4. **토지·건축 / 부동산** — 개발행위·농지전용, 토지이용·건축 인허가
5. **법인 설립 / 등록** — 법인·사업자, 협동조합·비영리단체 설립
6. **손실보상 / 민원** — 보상금 협의·이의, 진정·민원 작성

### 2-3. 메인 페이지 섹션 → 보일러플레이트 컴포넌트 매핑

| 순서 | 섹션 | 보일러플레이트 컴포넌트 | 내용 |
|---|---|---|---|
| 1 | Hero | `Hero` 또는 `ParallaxHero` | 슬로건 + 무료상담 CTA + 신뢰 한 줄(상담/경력) |
| 2 | 신뢰 띠 | `TrustBand` (marquee) | "무료상담 · 비밀보장 · 명확한 비용 · 신속처리" |
| 3 | 업무분야 | `Features` | 6개 분야 카드(아이콘+제목+설명) |
| 4 | 사무소 강점/프로세스 | `Process` | 상담→검토→서류작성→접수→완료 4~5단계 |
| 5 | 신뢰 지표 | `Stats` | 누적상담 5,000+ / 만족도 98% / 경력 15년 / 분야 20+ |
| 6 | 구성원 | `Team` | 대표·소속 행정사 카드(hover bio) |
| 7 | 처리사례 | `Testimonials` 또는 카드그리드(요청 시 추가) | 최근 처리사례 + 더보기 |
| 8 | 자주 묻는 질문 | `FAQ` | 비용/기간/절차 관련 |
| 9 | 오시는 길 | `MapSection` | 주소/전화/영업시간 + 지도 iframe |
| 10 | CTA + 문의 | `CTA` + `Contact` | 전화/카카오/폼 |
| 고정 | 플로팅 버튼 | `FloatingCta` | 전화 + 카카오 상담 |

> 별도 페이지: `사무소 소개`, `업무분야 상세`, `처리사례 목록/상세`, `오시는 길` → 라우트 추가로 구성.

## 3. 컬러 후보 (오늘 전달용)

미리보기: **`design/color-candidates.html`** (브라우저로 열면 6안을 실제 목업에 적용해 비교 가능)

| 안 | 이름 | 포인트 | 성격 | 레퍼런스 유사도 |
|---|---|---|---|---|
| **A** | 베르밀리온 | `#E5493A` | 따뜻·강단 | ★★★ (레퍼런스 매치) |
| **B** | 딥네이비 × 골드 | `#1B3A66` / `#C2A35B` | 공신력·권위 | ★ |
| **C** | 테라코타 × 토프 | `#B65A3C` / `#7A6A57` | 휴머니즘·부드러움 | ★★★ |
| **D** | 딥그린 × 골드 | `#1F4A38` / `#C2A35B` | 안정·신뢰 | ★★ |
| **E** | 버건디 × 웜그레이 | `#7C2D38` | 격식·프리미엄 | ★★ |
| **F** | 슬레이트블루 × 코퍼 | `#2C4A6E` / `#BE7B4F` | 모던·클린 | ★ |

**추천**: 레퍼런스 톤을 유지하려면 **A(베르밀리온)** 또는 **C(테라코타)**, "관공서·행정 공신력"을 더 강조하려면 **B(딥네이비×골드)**.

### 적용 방법
선정된 안의 색을 `src/app/globals.css` 의 `:root` 변수(oklch)로 변환해 교체 → 헤더/버튼/푸터 등 전체 일괄 반영.
주요 변수: `--primary`(포인트), `--primary-foreground`, `--background`, `--foreground`, `--muted-foreground`, `--border`, `--radius`.

### 블로그 상세 (2-b)
- 글을 **홈페이지 관리자에서 복붙 등록 → Supabase `posts` 테이블 저장 → 홈페이지 자체 도메인에 게시** (구글·홈페이지 SEO 유리)
- 네이버 원본과 동일 내용일 경우 `canonical` 처리로 중복 페널티 회피
- **관리자 페이지(`/admin`)는 후순위** — 초기엔 데이터 직접 시드/SQL 입력, 추후 작성·관리 UI 구축
- 표시 측(블로그 목록 카드 + 상세 페이지)은 먼저 구축, 쓰기 UI는 나중에
